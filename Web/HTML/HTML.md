# HTML
--------------------------------
```
Hypertext Markup Language
HT - HyperText, 문서와 문서가 링크로 연결되어 있다.
M - Markup, 태그로 이루어져 있다.
L - Language

-> 웹페이지를 만들기 위한 언어로 웹브라우저 위에서 동작하는 언어.
```
